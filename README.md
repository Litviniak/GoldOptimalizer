# GoldOptimalizer

Projekt ma na celu stworzenie addona, który pozwoli na optymalizację zdobywania złota względem czasu poświęconego

* UI

  UI zostanie stworzonę z pomocą plików XML
* Integracja z serwisem zewnętrznym

  Integracja poprzez SDK z grą World of Warcraft, możliwość integracji z serwisem raider.io
  
* Baza danych i lokalna konfiguracja zawarta w plikach gry

* Czas rzecziwisty:

  Sprawdzanie ekwpipunku,
  Przeliczanie obecnych w nim materiałów na przedmioty,
  pokonanych i jeszcze nie pokonanych bossów danej instancji i w przypadku pokonania wyświetlenie rzeczy które z nich otrzynamo,
  śledzenie zdobytego złota w jednostyce czasu wybranej przez gracza i śledzenie ilości złota na instancje
